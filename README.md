## Build on local
Step 1: create a file `.env` and copy the below varibales to the file:
```
# bestbuy API base URL
VUE_APP_BESTBUY_API_URL=https://api.bestbuy.com/v1/

# bestbuy API_KEY
VUE_APP_API_KEY=mPlbr5GXMVkagVgzwT7T2V5X
```
Step 2: then build
```
npm i
npm run server
```
